# Django Ecommerce Platform

A scalable e-commerce backend platform built with Django and Django REST Framework.  
The project provides RESTful APIs for authentication, product management, cart functionality, orders, and user management.

---

## Features

- JWT Authentication
- User Registration & Login
- Product Management
- Category Management
- Shopping Cart System
- Order Management
- RESTful API Architecture
- Secure Backend Structure
- API Serialization & Validation

## Installation

Clone the repository:

```bash
git clone https://github.com/AB-Git23/django-ecommerce-platform.git
cd django-ecommerce-platform
```

Create virtual environment:

```bash
python -m venv venv
```

Activate virtual environment:

### Windows
```bash
venv\Scripts\activate
```

### Linux / Mac
```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start development server:

```bash
python manage.py runserver
```
